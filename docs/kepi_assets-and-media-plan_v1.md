# 《客批》素材与媒体计划 v1

> 目标：把游戏静态素材的产出范围、优先级、规格、出图 prompt 与降级方案一次定清楚
> 依据：[美术风格设定 v1](kepi_art-style-design_v1.md)
> 更新：2026-06-25 — 合并角色 / 敌人出图 prompt 清单

## 1. 素材总原则

- 素材优先服务主流程，不做无关装饰
- 先做能跑通一局的核心素材，再补可选表现层
- 所有素材都要适配网页端和黑客松 demo 节奏
- 断网可玩，因此关键素材必须本地可用
- 结局相关素材需要支持本地兜底，不依赖在线资源
- 美术风格统一遵循 [美术风格设定 v1](kepi_art-style-design_v1.md)

## 1.1 风格基线

- 场景：温润写意的东方奇幻水彩，参考《大鱼海棠》的气质，但保留客家土楼识别度
- 人物：2D 动画赛璐璐感，线条干净，情绪温和，服饰层级清晰
- 敌人：与我军同画风，但更冷硬、器物成精、青灰朱红压迫感更强
- 土楼：圆形土楼是主识别点，必须一眼可见
- 画面：不写实、不 3D，不做无关装饰堆叠

### 2.2A 出图规则

- 我军统一走「温润、蓝靛、土布、写意」的客家人物风格
- 族长、风水先生、农夫、守卫的服饰层级要一眼可辨
- 水客强调「挑担收信」，乡贤强调「修家园的体面感」
- 敌军统一走「冷硬、器物成精、青灰 + 朱红」的反派感
- 敌人必须让人看出它阻挡的是哪段返乡历史

### 2.2B 出图优先级

1. **族长**、**风水先生**（造型最复杂，最先验证画风）
2. 农夫、守卫、水客、教书先生、乡贤
3. 遗忘军团 6 敌（同画风 + 冷硬异质感）

### 2.2C 通用风格基底

每个 prompt **最前面**粘贴以下段落。

#### 中文版（即梦 / 可灵）

> 《大鱼海棠》风格，温润写意的国风手绘水彩，2D 动画赛璐璐上色，柔和的手绘笔触，暖色调氛围光，木质围屋室内的温暖逆光，质感细腻不写实不 3D，人物面部柔和、眼神温润有神，线条干净流畅，整体气质宁静、有东方诗意和年代感，半身立绘，纯色或虚化背景，角色居中

#### 英文版（Midjourney）

> in the art style of Chinese animated film "Big Fish & Begonia", warm painterly 2D cel-shaded illustration, soft hand-drawn watercolor texture, gentle warm ambient lighting, traditional Hakka roundhouse wooden interior with warm backlight, serene poetic East-Asian mood, clean flowing linework, soft tender facial features with luminous eyes, character portrait half-body, simple blurred background, --ar 3:4 --niji 6

### 2.2D 敌军额外异质感（6 敌共用）

在通用基底后追加：

> 色调偏冷硬青灰与朱红，器物成精的诡异感，与人物角色的温暖人情味形成对比，反派氛围

若出图过于可爱，加重：`狰狞、压迫、menacing、诡异`

## 6. 资源规格建议

### 6.1 图片

- 建议使用 `PNG` 或 `WebP`
- 关键立绘尽量透明底
- 棋盘和背景图优先横向大图
- UI 图标可统一风格

### 6.2 音频

- 建议使用 `MP3` 或 `WAV`
- BGM 和循环环境音要控制体积
- 客家话朗读尽量按信件拆条

### 6.3 命名建议

```txt
public/
  images/
    board/
    characters/
    enemies/
    ui/
    ending/
  audio/
    bgm/
    sfx/
    voice/
```

## 7. 降级策略

- 土楼三阶段吃紧时，可降成两阶段
- 棋子 / 敌人立绘不足时，可先用统一风格占位图
- 结局信件不足时，先保证真实侨批原文优先
- 朗读音频不足时，先保证关键结局信有声音
- 视觉特效不足时，不影响主流程

## 8. 与开发阶段的关系

- Phase 0：素材目录、命名、出图 prompt 与风格锚点已定
- Phase 4：接入棋盘、UI、立绘、特效
- Phase 5：接入结局与音频
